# mkgrade-resource

本仓库用于归档、整理和发布多个独立的 HTML 页面。这些页面不针对任何特定域名或具体网站，仅作为通用资源页面进行存放与维护。

## 项目简介

`mkgrade-resource` 是一个以 HTML 页面为主要内容的归档仓库。页面内容涵盖信息展示、工具界面、说明文档等多种类型，均以独立、自包含的 HTML 文件形式存在，便于直接查看或嵌入使用。

仓库不对外提供任何形式的服务承诺或内容担保，仅作为静态资源存放与版本管理之用。

## 目录说明

```
mkgrade-resource/
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的静态资源（CSS、JS、图片等）
└── README.md       # 本文件
```

- `pages/`：每个 HTML 文件为一个独立页面，命名清晰，内容自包含。
- `assets/`：按需存放页面所需的公共或私有资源，保持目录结构整洁。

## 页面归档说明

- 所有页面均为独立 HTML 文件，可直接在浏览器中打开。
- 页面之间无依赖关系，可单独使用或引用。
- 页面内容可能随时间更新或移除，请以仓库当前状态为准。
- 归档页面不保证兼容所有浏览器或设备，仅提供一般性适配。
